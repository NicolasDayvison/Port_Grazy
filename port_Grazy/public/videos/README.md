# Vídeos do Portfólio

## Como adicionar vídeos:

1. **Coloque seus vídeos nesta pasta** (`port_Grazy/public/videos/`)
2. **Formatos suportados:** MP4, WebM, OGV
3. **Tamanho recomendado:** 9:16 (vertical) para melhor visualização no celular

## Exemplos de nomes de arquivo:
- `video-moda-1.mp4`
- `video-maternidade-1.mp4`
- `video-beleza-1.mp4`
- `video-bem-estar-1.mp4`
- `video-lifestyle-1.mp4`

## Como referenciar no código:
```html
<video controls poster="https://placehold.co/300x533/E8A0BF/FFFFFF?text=Moda+1">
    <source src="/videos/video-moda.mp4" type="video/mp4">
    Seu navegador não suporta vídeos.
</video>
```

## Dicas:
- Use nomes de arquivo simples (sem espaços ou caracteres especiais)
- Mantenha os vídeos otimizados para web (tamanho < 10MB)
- Adicione posters (thumbnails) para melhor carregamento
